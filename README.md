# ShieldCheck - Password Security Checker 🔐

A simple cybersecurity web tool that analyzes password strength in real time.

The application helps users understand how secure their passwords are by evaluating common security criteria such as length, character diversity, and complexity.

---

## 🌐 Live Demo

👉 https://metodo-steel.vercel.app/

---

## 📌 Features

* Real-time password strength analysis
* Security indicators for:

  * Password length
  * Uppercase letters
  * Numbers
  * Special characters
* Visual strength feedback (Weak / Medium / Strong)
* Simple and clean user interface

---

## ⚙️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 Purpose of the Project

This project was created as part of a personal learning portfolio focused on **cybersecurity and web development fundamentals**.

The goal was to explore how password security rules can be evaluated programmatically and presented in a clear interface for users.

---

## 📂 Project Structure

ShieldCheck/

index.html  (HTML, CSS and JavaScript in a single file)

For simplicity and easy deployment, the entire project is implemented in a single HTML file that includes embedded CSS and JavaScript.

This approach makes the application lightweight and easy to host as a static webpage.
